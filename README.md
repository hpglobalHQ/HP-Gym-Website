# HP Gym Website 🏋️‍♂️

A clean and responsive multi-page gym website built using **HTML5 and CSS3**. This project represents a fitness website for **HP Gym**, featuring membership plans, schedules, contact information, and gym details in a simple and user-friendly design.

## About The Project

This project was created to provide a modern-looking website for a gym business. It focuses on clean UI, easy navigation, and responsive layouts without using any frameworks or external libraries.

The website includes multiple pages connected through a shared navigation system and consistent styling.

## Features

* Multi-page website structure
* Responsive design for different screen sizes
* Modern gym-themed interface
* Membership plans section
* Weekly schedule page
* About page with gym information
* Contact page for enquiries
* Shared navigation across all pages
* Reusable CSS styling

## Project Structure

```bash
HP-GYM-Wensite/
│
├── index.html        # Homepage
├── about.html        # About the gym
├── plans.html        # Membership plans
├── schedule.html     # Gym schedule and timings
├── contact.html      # Contact information
└── style.css         # Global styling
```

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## How To Run

1. Download or clone the project.
2. Open the project folder.
3. Run `index.html` in your browser.

No installation or dependencies are required.
